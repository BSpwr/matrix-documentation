Overview

HAL is the lowest-level abstraction for the MATRIX Creator drivers. You can interface directly with HAL, or use the higher level components like CORE, and MATRIX OS.

Tooling

  • xc3sprog - To program the FPGA.
  • OpenOCD - To program the microcontroller.
