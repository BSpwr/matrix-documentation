Sensor Tests

With the following instructions you will be able to run demos for the Everloop LED Array and for the IMU. These demos make use of the hardware abstraction layer. If you followed the getting started steps the FPGA and the SAM3 MCU should be programmed and ready. Also, SPI should be enabled.

Thus you can get the required source code and build it:

git clone https://github.com/matrix-io/matrix-creator-hal.git
cd matrix-creator-hal
mkdir build && cd build
cmake .. && make

If the project built with no issues you can run the demos. The demos run for ever, you can terminate them with Control+C.
