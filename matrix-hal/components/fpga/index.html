FPGA

Once you have installed the required software you can program the FPGA. This will allow you to play with the everloop LED array.

First, get the firmware.

wget http://packages.matrix.one/matrix-creator-firmware/firmware-0.8.tar.gz
tar xzvf firmware-0.8.tar.gz

Then flash the firmware using the xc3sprog program that should be installed on the Raspberry.

sudo xc3sprog -c matrix_pi firmware-0.8/blob/system.bit -p 1

Sample result:
